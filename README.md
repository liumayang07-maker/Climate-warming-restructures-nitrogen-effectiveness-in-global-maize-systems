# Nitrogen management meta-analysis public release

This repository is a private staging area for reviewer- and publication-ready public materials from the nitrogen management meta-analysis project.

The repository is intentionally kept private while files are screened. It can be switched to public after scripts, source data, figures, and documentation have been checked for reproducibility, licensing, and sensitive information.

## Planned contents

- Reproducible analysis scripts
- Figure-generation scripts
- Public source-data tables or de-identified example data
- Environment and dependency notes
- Reviewer-facing documentation

## Excluded from public release

- Draft manuscripts and cover letters
- Internal review notes
- Raw credentials or local environment files
- API keys, tokens, and `.cdsapirc`
- Local virtual environments
- Copyright-restricted literature PDFs or third-party images
