# Figure6

This directory is the public-release package for the current `Figure 6` workflow.

## Scope

Figure 6 compares candidate warming-adaptive management strategies against regional conventional-practice references under pre-flowering risk (PFR) and grain-filling risk (GFR). The public package is limited to the scripted grain-yield strategy-comparison branch and the final Source Data CSV exports.

The public workflow includes:

- PFR strategy-comparison yield distributions for Figure 6B
- PFR Games-Howell mean-yield differences for Figure 6C
- GFR strategy-comparison yield distributions for Figure 6D
- GFR Games-Howell mean-yield differences for Figure 6E
- Dunn tests with Benjamini-Hochberg adjustment for rank-based distributional differences
- group-level mean, standard-deviation and sample-size summaries

## Packaged Input Data

- `input_data/adaptation_strategy_dataset.csv`

This is the minimum CSV input used by `scripts/compare_adaptation_strategies_yield.R`. The local `source_table.xlsx` file is not included because it is not required by the public R script.

## Packaged Output Data

`output_data/` contains final Source Data-aligned CSV exports and script-generated numerical summaries:

- `Figure6_data_set.csv`
- `Figure6c_games_howell_mean_differences.csv`
- `Figure6e_games_howell_mean_differences.csv`
- `Figure6b_*_from_script.csv`
- `Figure6c_games_howell_mean_differences_from_script.csv`
- `Figure6d_*_from_script.csv`
- `Figure6e_games_howell_mean_differences_from_script.csv`

The non-suffixed `Figure6c` and `Figure6e` files are exact Source Data block extracts. Files ending in `_from_script.csv` are generated by rerunning the public R script and are kept separate so that reruns do not overwrite the final Source Data extracts.

## Exclusions

- Figure image files and editable design files (`*.pdf`, `*.ai`, `*.jpg`, `*.svg`, `*.opju`) are not included.
- The full manuscript Source Data workbook is not included.
- The local `source_table.xlsx` helper file is not included.

## Reproducibility Notes

- The public R script uses repository-relative paths.
- Rerunning the script writes `_from_script.csv` numerical outputs and temporary panel PDFs; PDFs are intentionally excluded from the GitHub package.
- Final Figure 6C and Figure 6E rendering was completed in Origin 2025, so the public package provides the numerical Source Data and scripted statistical summaries rather than editable figure-layout files.
